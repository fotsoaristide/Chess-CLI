# Chess Tournament Manager (CLI)

## 📌 Description

This project is a command-line application that allows you to manage chess tournaments.

You can:

* Create tournaments
* Add players
* Generate rounds
* Enter match results
* Display final rankings

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/fotsoaristide/Chess-CLI.git
cd projet_4
```

### 2. Create virtual environment

```bash
python -m venv env
```

### 3. Activate environment

**Windows:**

```bash
env\Scripts\activate
```

**Mac/Linux:**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the application

```bash
python main.py
```
